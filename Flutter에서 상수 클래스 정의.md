# 1. 상수 클래스의 정의
# 상수 클래스란 프로그램 전체에서 **공통으로 사용되는 변하지 않는 값(상수)**들을 한곳에 모아 관리하는 클래스를 의미합니다.


# 2. 왜 사용하는가? (주요 장점)
# 가독성 및 유지보수성 향상
# 중복되는 코드를 상수 클래스로 모아서 한번에 수정할수 있습니다.

# 성능 최적화
# 상수클래스를 사용하여 중복 코드가 여기저기 있지 않고 하나의 메모리 주소로 모이게 되어 
# 메모리의 성능이 향상됩니다.

# 네임스페이스(Namespace) 제공
# 단순히 전역 변수를 선언하는 것보다, AppColors.primary, AppStrings.loginError처럼 클래스 이름을 통해 해당 상수의 **맥락(Context)**을 즉시 파악할 수 있습니다.
