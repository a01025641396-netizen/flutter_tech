8. Flutter 앱에서 "https://google.com" 요청 시 과정
이 질문은 네트워크 계층 전반에 대한 이해를 묻습니다.

DNS Lookup: Flutter의 HTTP 클라이언트(dio, http 등)가 google.com의 IP 주소를 얻기 위해 DNS 서버에 질의합니다.

TCP 3-Way Handshake: IP 주소를 확보하면 전송 계층에서 서버와 연결을 설정합니다.

TLS Handshake (HTTPS 핵심): SSL/TLS 인증서를 확인하고, 데이터를 암호화할 '대칭키'를 안전하게 교환합니다.

HTTP Request 전송: 암호화된 터널을 통해 GET/POST 요청을 보냅니다.

서버 처리 및 Response: 구글 서버가 요청을 해석하고 HTML이나 JSON 데이터를 반환합니다.

Rendering: Flutter 앱은 수신한 데이터를 바탕으로 화면을 갱신합니다.
